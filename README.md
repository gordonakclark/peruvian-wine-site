# 🍷 Taste Peru – Peruvian Wine Website

This is a simple one-page website designed to showcase and test a business idea for selling Peruvian wine online.

It is beginner-friendly, fully static (no backend required), and can be hosted for free using GitHub Pages.

---

## 🚀 Features

* Clean landing page design
* Featured wine products section
* About section
* Contact form (demo only)
* Mobile responsive layout

---

## 📁 Project Structure

```
index.html   # Main website file
README.md    # Project documentation
```

---

## 🧑‍💻 How to Use

### 1. Download or Copy Files

Save the HTML file as:

```
index.html
```

### 2. Open Locally

Double-click the file or open it in a browser to preview.

### 3. Upload to GitHub

* Create a new repository
* Upload `index.html`

### 4. Enable GitHub Pages

* Go to **Settings → Pages**
* Select **Deploy from branch (main)**

Your site will be live at:

```
https://yourusername.github.io/repository-name/
```
